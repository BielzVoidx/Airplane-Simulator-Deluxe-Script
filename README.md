✈ Deluxe God Auto Contract – Airplane Simulator

Deluxe God Auto Contract is an advanced automation script designed for
Airplane Simulator that automatically flies your aircraft to active
contract locations using smooth and controlled flight movement.

The script provides a clean in-game GUI, stability-focused flight logic,
and safety reminders to ensure proper usage before activating
automation.

  ----------
  FEATURES
  ----------

-   Auto Contract Flight Automatically travels to selected contract
    markers. Smooth tween-based flight system. Smart distance and target
    checks.

-   Deluxe GUI Interface Draggable modern UI. Enable / Disable toggle
    button. Status indicators.

-   Safety Warning System Blinking reminder message. Informs users to:

    -   Start the airplane engine first.
    -   Select contracts manually.

-   Interactive Feedback Sound effect when toggling Auto Contract.

-   God Flight Movement Ascend → Travel → Landing sequence. Stable
    positioning and arrival detection.

  ------------
  HOW TO USE
  ------------

1.  Spawn and enter your airplane.
2.  Start the airplane engine manually.
3.  Select contracts manually in-game.
4.  Press ENABLE in the GUI.
5.  The script will automatically fly to contract locations.

  -----------------
  IMPORTANT NOTES
  -----------------

-   The engine must be running before enabling Auto Contract.
-   Contracts are not auto-selected.
-   Designed for smooth and stable automated travel.

  --------------------
  TECHNICAL OVERVIEW
  --------------------

-   Built using Roblox Lua.
-   TweenService-based movement system.
-   Target validation and anti-repeat logic.
-   Lightweight client-side execution.
