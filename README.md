✈ DELUXE GOD AUTO CONTRACT
Airplane Simulator – README
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Deluxe God Auto Contract is an advanced automation script designed for
Airplane Simulator that automatically flies your aircraft to active
contract locations using smooth, controlled, and stability-focused
flight movement.

The script includes a modern in-game GUI, automated navigation logic,
and safety reminders to ensure correct usage before activating
automation.

━━━━━━━━━━━━━━━━
🚀 FEATURES
━━━━━━━━━━━━━━━━

✦ Auto Contract Flight
• Automatically travels to selected contract markers.
• Smooth TweenService-based flight movement.
• Smart distance validation and target detection.
• Anti-repeat destination logic.

✦ Deluxe GUI Interface
• Clean and modern draggable UI.
• Enable / Disable toggle button.
• Status indicators for script state.

✦ Safety Warning System
• Blinking reminder message inside the GUI.
• Warns users to:
  - Start the airplane engine first.
  - Select contracts manually.

✦ Interactive Feedback
• Sound effect when toggling Auto Contract.

✦ God Flight Movement
• Ascend → Travel → Landing flight sequence.
• Stable positioning and arrival detection.

━━━━━━━━━━━━━━━━
📖 HOW TO USE
━━━━━━━━━━━━━━━━

1. Spawn and enter your airplane.
2. Start the airplane engine manually.
3. Select contracts manually in-game.
4. Press ENABLE in the GUI.
5. The script will automatically fly to contract locations.

━━━━━━━━━━━━━━━━
❗ IMPORTANT
━━━━━━━━━━━━━━━━

• The script is NOT perfect and may occasionally make mistakes.
• Automation can fail after long usage sessions.
• If the script starts failing repeatedly, complete ONE contract manually.
• Performing a manual contract usually restores normal behaviour.

━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━
